# cmd速查表(DOS命令)

| 命令提示符         |                                          |
| ------------------ | ---------------------------------------- |
| 切换盘符           | `盘符名称：`                             |
| 进入文件夹         | `cd  <filename>`                         |
| 进入多级文件夹     | `cd <file1name>\<file2name>\<file3name>` |
| 返回上一级         | `cd ..`                                  |
| 直接回根目录       | `cd \`                                   |
| 查看当前文件夹内容 | `dir`                                    |
| 清屏               | `cls`                                    |
