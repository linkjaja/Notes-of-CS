## Last but not least

* 文档命名规则：英文小写，使用短横杠连接，大类写在前。

* commit 符号说明：

  * 调色板:art: -- art -- 改进代码结构/文件夹结构（美观）

  * 小铅笔:pencil2:--pencil2 -- 修复错误，包括typo（修）

  * 大铅笔:pencil:-- pencil -- 提交文档（增）

  * 火焰:fire:-- fire -- 移除代码或文件（删）
